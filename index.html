Stacks üìö

A stack is a data structure that follows the Last In First Out (LIFO) principle. The last element added to the stack is the first one to be removed.

Uses of Stack

  • Used in function calls and recursion in programming languages.
  • Helps in undo operations in text editors (e.g., CTRL+Z).
  • Used in web browsers for navigation (back and forward buttons).
  • Memory management, especially in allocating space for recursive functions.

Real-World Applications üåé

  • Expression Evaluation: Used for evaluating mathematical expressions in compilers.
  • Backtracking Algorithms: In algorithms like Depth-First Search (DFS), the stack is used to keep track of the vertices.
  • Undo Feature in Applications: Keeps track of operations for undo functionality in text editors.

Example Stack Operations ‚öôÔ∏è

  • Push: Adds an item to the top of the stack.
  • Pop: Removes the top item from the stack.
  • Peek: Views the top item without removing it.
  • Clear: Removes all items from the stack.

    #define MAX 10
    int stack[MAX];
    int top = -1;

    void push(int val) {
      if (top >= MAX - 1) {
        printf("Stack Overflow\n");
      } else {
        stack[++top] = val;
        printf("%d pushed to stack\n", val);
      }
    }

    void pop() {
      if (top == -1) {
        printf("Stack Underflow\n");
      } else {
        printf("%d popped from stack\n", stack[top--]);
      }
    }

    void peek() {
      if (top == -1) {
        printf("Stack is empty\n");
      } else {
        printf("Top element is %d\n", stack[top]);
      }
    }
